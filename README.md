# 🛡️ MERN Authentication

Welcome to the **MERN Authentication** project!
A full-featured authentication system built using the MERN stack that provides secure and scalable user management. 

This project implements complete authentication flows such as registration, login, email verification, and password recovery.

## Project Overview

This project is designed to demonstrate a real-world authentication system used in modern web applications.

### It focuses on:
- Secure user authentication using JWT  
- Cookie-based session handling
- Email verification workflow  
- Password reset functionality

The system follows best practices for security and scalability, making it suitable for production-level applications.

## Features

- Secure user authentication with JWT and cookie-based sessions 
- Password hashing using bcrypt for enhanced security  
- Email verification and password reset system
- Protected routes with middleware and token validation  
- Clean, scalable REST API with modular architecture 

## Tech Stack

- **Frontend**:
  - Javascript
  - React
  - React Router Dom
  - Axios
  - React Toastify
  - Tailwind CSS

- **Backend**:
  - Node.js
  - Express.js
  - MongoDB + Mongoose
  - JWT (Authentication)
  - Bcrypt.js (Password hashing)
  - Nodemailer
